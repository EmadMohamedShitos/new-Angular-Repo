Section 2 ‚Äî For Linux OS (Ubuntu Example)

Docker Installation on Linux

Step 1 ‚Äî System Update
sudo apt update
sudo apt upgrade -y
Step 2 ‚Äî Remove Old Versions
sudo apt remove docker docker-engine docker.io containerd runc
Step 3 ‚Äî Install Required Packages
sudo apt install ca-certificates curl gnupg lsb-release -y
Step 4 ‚Äî Add Docker GPG Key
sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
Step 5 ‚Äî Add Docker Repository
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] \
https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | \
sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
Step 6 ‚Äî Install Docker
sudo apt update
sudo apt install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin -y
Verify Installation
sudo docker --version

Section 3 ‚Äî For Windows OS

Docker Installation on Windows

  1. Open PowerShell as Administrator.
  2. Install Windows Subsystem for Linux (WSL): 
    wsl --install
  3. Go to Docker Desktop for Windows .
  4. Download and install Docker Desktop.
  5. After installation, open PowerShell or CMD and verify: 
    docker --version

Section 4 ‚Äî Docker Desktop (GUI) for Linux

Docker Desktop for Ubuntu GUI

Follow the official installation guide:

https://docs.docker.com/desktop/install/ubuntu/

Or install manually:
curl -LO https://desktop.docker.com/linux/main/amd64/docker-desktop-latest.deb
sudo apt install ./docker-desktop-latest.deb
systemctl --user start docker-desktop
